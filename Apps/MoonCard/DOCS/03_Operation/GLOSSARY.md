# GLOSSARY.md

## ハワイ語表記のルール

- **マクロン（長音記号）**：`ā` `ē` `ī` `ō` `ū` のように母音の上に線を引く表記。長母音を示す
- **オキナ**：`ʻ`（開き一重引用符に似た記号）。声門閉鎖音を示す、ハワイ語アルファベットの正式な文字の一つ
- 本プロジェクトでは、`name_en` フィールドに必ずこの正式表記を用いる（例：`Ku-kahi` ではなく `Kū-kahi`、`Ole-pau` ではなく `ʻOle-pau`）
- 表記ゆれが発生した実例は `DATA_AUDIT.md` を参照

## Alias について

Aliasは、その用語について**実際に会話・資料の中で使われた呼び方**を記録する欄である。
存在しない呼び方を、検索性のために創作して登録することはしない。
