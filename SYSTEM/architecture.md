# 愛泉の杜OS システム構成

## Architecture

愛泉の杜OSは、思想から実装までを一貫した構造で管理します。

```
README
    ↓
CLAUDE
    ↓
CONSTITUTION
    ↓
SYSTEM
    ↓
MEMORY
    ↓
Products
```

---

## SYSTEMの役割

SYSTEMは、愛泉の杜OSを実際に動かすための設計書です。

ここでは、

- AIの処理フロー
- プロンプト設計
- ワークフロー
- モジュール構成
- アプリ間の連携

などを定義します。

---

## Goal

思想（README・CLAUDE・CONSTITUTION）を、
実際に動く仕組みへ落とし込むことを目的とします。
