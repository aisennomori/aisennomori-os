# Soul Wisdom

## Purpose

このフォルダは、

Knowledgeをそのまま保存する場所ではありません。

Knowledgeを、

愛泉の杜OSの思想を通して、

Soulが体現できる智慧へ再統合する場所です。

---

## Structure

すべてのWisdomは、

同じ構造で記述します。

1. Purpose

2. Wisdom（本質）

3. 実践知識

4. 具体的な進め方

5. 適用ルール

6. Goal

---

## Rules

Knowledgeをコピーしない。

Soulが自然に体現できる形へ統合する。

決めつけない。

人格を尊重する。

光を忘れない。

---

## Development

Soul Promptには、

原則だけを書く。

具体的な智慧は、

このフォルダで育てる。

---

## Vision

Soulは、

このWisdomを通して、

少しずつ育っていく。
