# DESIGN_RULES.md

## 世界観

深い夜空の紺と、月光を思わせる金。
愛泉の杜の既存アセット（金と黒の月相アイコン）の世界観を踏襲しつつ、
「静かに向き合う」体験を邪魔しない、落ち着いたトーンを基調とする。

## タイポグラフィ

- 見出し・名前：`Shippori Mincho`（明朝体、風格と静けさを表現）
- 本文・UI：`Zen Kaku Gothic New`（可読性重視のゴシック体）
- 文字サイズは全体的に控えめ（10〜23px）に抑え、情報密度を上げつつも圧迫感を出さない

## レイアウト

- カード：角丸22px、影あり、最大幅460px（モバイルの手のひらサイズを基準）
- データタイル：角丸13px、2列グリッド、薄い金の枠線＋わずかな金の背景トーン
- 箇条書きは「・」ではなく、金色の小さなドット（`::before`）で表現し、静けさのあるアクセントにする

## 画像の扱い

- カード本体画像はサムネイル（76×76px、角丸14px）として控えめに配置し、テキスト情報を主役にする
- 画像未着手の夜は、装飾のない準備中アイコンのみを表示し、無理に埋めない

## モーション

- カード送りのトランジションのみ（0.45秒、`cubic-bezier(.65,.05,.36,1)`）
- 過度なアニメーションは使わない。「静かにめくる」感覚を損なわないため

## 命名規則

- 画像フォルダは `moon_card_images/`（既存の `画像`／`moon_01.png`〜`moon_30.png` と衝突しないよう意図的に区別。理由は `DECISIONS.md` 参照）
- 画像ファイル名は `card_XX.jpg`（`XX`は2桁ゼロ埋め、`day`と対応）

## 他ツールとのアクセントカラーの使い分け（moon-tools全体）

月読みツール集トップページ（`moon_tools.html`）では、ツールごとに異なるアクセントカラーを割り当てている。

| ツール系統 | アクセント |
|---|---|
| moon（青系＋金） | `#6090e0` 〜 `#c9a84c` |
| mooncal（緑青系＋橙） | `#4F9D94` 〜 `#E2A23C` |
| mooncycle（緑系） | `#8FAE7E` 〜 `#4F9D94` |
| **moongallery（月読みカード）** | `#d4608a`（ピンク）〜 `#c9a84c`（金） |

新しいツールを追加する際は、既存の配色と重複しない新しい組み合わせを選ぶこと。
