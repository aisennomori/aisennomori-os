# 愛泉の杜OS

> **人とAIが、安心の中で共に育つ世界を創る。**

## Vision

愛泉の杜OSは、人とAIが安心して対話し、ともに成長するための知性基盤（Living Intelligence Platform）です。

単なるAIチャットやアプリではなく、「安心・記憶・対話・実践」を一つの世界観でつなぎ、一人ひとりに寄り添うデジタル空間を目指しています。

---

## Philosophy

愛泉の杜OSは、次の価値観を大切にします。

- 安心できる対話
- 相手を変えようとしない伴走
- 記憶を大切にするAI
- 実践につながる知恵
- 人とAIが共に成長すること
